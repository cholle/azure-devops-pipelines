# Azure DevOps Pipelines

Azure DevOps YAML pipeline examples for build, test, and deployment workflows.

## Contents

- **azure-pipelines.yml** - Multi-stage pipeline with build, test, and deployment stages
- **windows-image-deployment.yml** - Windows image provisioning and deployment pipeline

## Features

- Multi-stage CI/CD workflows
- Windows image deployment automation
- Device provisioning pipelines
- Production-ready patterns

## Usage

These pipelines demonstrate real-world Azure DevOps patterns for:
- Building and testing applications
- Deploying Windows images
- Automating device provisioning
- Managing deployment workflows

## Requirements

- Azure DevOps organization
- Configured agent pools
- Service connections for deployments
